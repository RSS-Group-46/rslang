# RS Lang
Project name is a `RS Lang`. This application is for learning foreign words with interval repetition techniques, tracking individual progress and mini-games.

****
## How to run project
* Make `git clone`
* Install dependencies `npm ci`
* Run webpack-dev-server `npm run start`
* Open url `localhost:8080` in your browser

## How to build project
* Make `git clone`
* Install dependencies `npm ci`
* Run webpack-dev-server `npm run build`
* Install http-server package `npm ci -g http-server`
* Run http-server `http-server dist`
* Open url `localhost:8080` in your browser
